\subsection{Loss function used}
\text{Dans le projet, on utilise comme loss fonction la MSE. 
Cependant puisque nous sommes dans le cadre d'une classification binaire, il serait plus approprié d'utiliser la Binary Cross-Entropy (BCE) comme fonction de perte.
 La BCE est mieux adaptée aux tâches de classification binaire car elle mesure la différence entre les distributions de probabilité des prédictions du modèle et des étiquettes réelles, ce qui permet une meilleure optimisation du modèle pour ce type de problème.}
